## Overview

Add session restoration to remember last viewed position. Users return to exactly where they left off - same folder, same image selected - without needing to re-navigate.

## Phases

### Phase 6: Session Restoration

**Goal**: Remember and restore last viewed image position on app relaunch
**Depends on**: Phase 5
**Plans**: 1 plan

Plans:
- [x] 06-01: Session state persistence (folder + image selection)

**Details:**
- Folder path persistence in `navigateToFolder()` to UserDefaults
- Selected image path persistence via `didSet` observer on `selectedID`
- Restore logic restores both folder and image on app launch
- Security check ensures restored folder path is within root directory
- Auto-scroll to restored image in file list
