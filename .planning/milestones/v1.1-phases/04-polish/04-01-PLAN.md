Refine FolderRowView and FolderTreeView to match Finder's native sidebar appearance:

1. **FolderRowView styling:**
   - Remove the Button wrapper - use Label directly for native list row behavior
   - Use `Label(node.name, systemImage: "folder")` for native folder icon styling
   - Apply `.foregroundStyle(.primary)` for text, let system handle folder icon color
   - Add `.onTapGesture` to the Label for folder navigation (single tap to navigate)

2. **DisclosureGroup refinements in FolderTreeView:**
   - The DisclosureGroup already provides native disclosure triangles - keep as-is
   - Ensure consistent indentation by not adding extra padding

3. **List styling consistency:**
   - Current `.listStyle(.sidebar)` is correct for Finder-like appearance
   - Ensure Section headers use sentence case ("Pastas", "Arquivos") - already correct

4. **Minor spacing adjustments:**
   - In folder header area, ensure the "Escolher Pasta" button uses standard macOS button styling
   - Path display should use `.font(.caption2)` for more subtle appearance

Note: Do NOT change the DisclosureGroup binding logic - it's working correctly for expansion persistence.
