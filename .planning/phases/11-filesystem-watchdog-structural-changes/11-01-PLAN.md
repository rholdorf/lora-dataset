**DirectoryWatcher.swift** -- Create a new file with a `final class DirectoryWatcher`:

1. Properties:
   - `private var source: DispatchSourceFileSystemObject?` -- the VNODE source
   - `private let url: URL` -- watched directory URL
   - `private let queue: DispatchQueue` -- dispatch queue for source events
   - `private var debounceWorkItem: DispatchWorkItem?` -- current pending debounce
   - `private let debounceDelay: TimeInterval` -- default 0.5s
   - `private let onChange: () -> Void` -- callback (fires on `queue` after debounce settles)

2. `init(url:queue:debounceDelay:onChange:)` -- store properties. `debounceDelay` defaults to 0.5.

3. `func start()`:
   - Guard `source == nil` (idempotent)
   - `let fd = open(url.path, O_EVTONLY)` -- event-only fd
   - Guard `fd >= 0`, log `[watchdog] failed to open fd for \(url.lastPathComponent)` on failure
   - Create `DispatchSource.makeFileSystemObjectSource(fileDescriptor: fd, eventMask: .write, queue: queue)`
   - `setEventHandler { [weak self] in self?.scheduleCallback() }`
   - `setCancelHandler { close(fd) }` -- MUST close fd in cancel handler
   - `resume()`
   - Store in `source`
   - `print("[watchdog] started watching \(url.lastPathComponent)")`

4. `private func scheduleCallback()`:
   - `debounceWorkItem?.cancel()`
   - Create new `DispatchWorkItem { [weak self] in self?.onChange() }`
   - `queue.asyncAfter(deadline: .now() + debounceDelay, execute: item)`
   - Store in `debounceWorkItem`

5. `func stop()`:
   - `debounceWorkItem?.cancel(); debounceWorkItem = nil`
   - `source?.cancel(); source = nil`
   - `print("[watchdog] stopped watching")`

6. `deinit { stop() }`

/// Removes a single cached entry by URL. Used for targeted eviction

    evict(url)
    print("[cache] removed \(url.lastPathComponent) on external deletion")
}
