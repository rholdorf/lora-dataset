  <behavior>
    ImageLoader:
    - loadImage(url:maxPixelSize:) returns NSImage? decoded via CGImageSourceCreateThumbnailAtIndex
    - Uses kCGImageSourceShouldCache: false on source creation (no raw data caching)
    - Uses kCGImageSourceShouldCacheImmediately: true on thumbnail (force decode now)
    - Uses kCGImageSourceCreateThumbnailWithTransform: true (respect EXIF orientation)
    - Returns nil for invalid/corrupt files

    ImageCacheActor:
    - image(for: URL) -> NSImage? returns cached image in O(1), moves entry to front of LRU order
    - insert(_: NSImage, cost: Int, for: URL) adds entry, evicts LRU if over budget
    - clear() removes all entries and resets totalCost to 0
    - evictToFraction(_ fraction: Double) evicts LRU entries until totalCost <= budget * fraction
    - Budget is Int(Double(ProcessInfo.processInfo.physicalMemory) * 0.15)
    - Cost accounting: each entry stores width * height * 4 byte cost
    - LRU eviction order: least-recently-accessed (touch on read) evicted first
    - Memory pressure: DispatchSource.makeMemoryPressureSource with .warning -> evictToFraction(0.5), .critical -> clear()
    - Uses source.data (not source.mask) in event handler
    - Contains `extension NSImage: @unchecked @retroactive Sendable {}` at file scope

    Tests (ImageLoaderTests):
    - testLoadsWithCGImageSource: loadImage with a known test image (create a 2x2 PNG in /tmp) returns non-nil NSImage
    - testReturnsNilForInvalidFile: loadImage with nonexistent URL returns nil

    Tests (ImageCacheActorTests):
    - testCacheHitReturnsCachedImage: insert an NSImage, image(for:) returns the same instance
    - testCacheMissReturnsNil: image(for:) on empty cache returns nil
    - testCostAccounting: insert entries, verify totalCost matches sum of width*height*4
    - testLRUEvictionOrder: fill cache past budget, verify oldest-accessed entry is evicted first
    - testEvictToFraction: insert entries, call evictToFraction(0.5), verify totalCost <= budget * 0.5
    - testClearRemovesAll: insert entries, clear(), verify image(for:) returns nil
    - testMemoryPressureWarning: call handleMemoryPressure(.warning), verify eviction to 50%
    - testMemoryPressureCritical: call handleMemoryPressure(.critical), verify cache is empty
